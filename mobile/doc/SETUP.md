# Configuração e Execução - Mobile

Este projeto é um aplicativo React Native desenvolvido com Expo.

## Pré-requisitos
- Node.js instalado
- Gerenciador de pacotes (npm ou yarn)
- Aplicativo **Expo Go** instalado no seu celular (Android ou iOS) ou um emulador configurado.

## Como Rodar
1. Navegue até a pasta `mobile`:
   ```bash
   cd mobile
   ```

2. Instale as dependências (caso ainda não tenha):
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npx expo start
   ```

4. Ler o QR Code:
   - Abra o app **Expo Go** no seu celular.
   - Escaneie o QR Code exibido no terminal.

## Estrutura de Pastas
- `src/screens`: Telas do aplicativo.
- `src/components`: Componentes reutilizáveis.
- `src/utils`: Utilitários (ex: armazenamento local).
- `assets`: Imagens e ícones.
