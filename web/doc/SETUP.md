# Configuração e Execução - Web

Este projeto é uma aplicação Web React (Vite).

## Pré-requisitos
- Node.js instalado
- Gerenciador de pacotes (npm ou yarn)

## Como Rodar
1. Navegue até a pasta `web`:
   ```bash
   cd web
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. Acesse no navegador:
   - Geralmente em `http://localhost:5173` (veja o terminal para confirmar).

## Estrutura
- `src/`: Código fonte React.
- `public/`: Assets estáticos.
